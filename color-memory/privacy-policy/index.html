Color Memory ("the App") is an Apple Watch game. We do not collect, store, or share personal information from users.

Data Collection

  • No account registration
  • No analytics SDK
  • No advertising SDK
  • No location tracking
  • No cloud sync

Local Data on Device

The App stores game ranking data (scores) locally on your Apple Watch/device using system storage only. This data is not transmitted to the developer.

Children's Privacy

The App does not knowingly collect personal information from children or any user.

Changes

We may update this policy if the App's features change. Updates will be posted on this page.

Contact

For questions, please use the support page: Color Memory Support.
